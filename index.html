HTML

  • The head element contains information that facilitates rendering of the webpage by the browser
  • Information in the head elemnt may include the following elements:
    • base
    • isindex
    • link
    • meta
    • nextid
    • range
    • style
    • title
  • Note that the title element is always required
  • The body element represents the visible content of an HTML document displayed to the user
  • There can be only one body element in a document

CSS

  • Acronym for cascading style sheets
  • CSS is used to format the layout of a web page, including design, styles, colours, and variations in display for different devices
  • 3 methods for adding CSS:
    • Inline by using the style attribute in an individual HTML element in body section
      Defines the style of a singular HTML element
    • Internal by using a style element in head section
      Defines the style of a singular HTML page
    • External by using a link element in head section to associate a separate .css file. Defines the style of multiple HTML pages
      rel attribute defines the relationship and href attribute defines the target
      • A css selector is used in the stylesheet to select the element to be styled with css properties
      • Frequently used css selectors include:
        • named element selector: header, footer, h1, h2, p, img, ul, ol for eg. p { text-align: center; color: red; }
        • id selector: css rule will be applied to an HTML element of a specific id = "mainNav", designated by a # for eg. #mainNav { text-align: center; color: red; }
        • class selector: css rule will be applied to an HTML element of a specific class = "drop-down-menu", designated by a . for eg. .drop-down-menu { left: 0; right: auto; }
        • universal selector; css rule will be applied to every HTML element, designated by a * for eg. * { margin: 0; padding: 0; }
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • Git is a free and open source distributed version control system
  • Git commands include:
    • git status: shows which branch you are currently on
    • git checkout -b [branch name]: creates a new branch and switches to it
    • git merge [branch name]: merge the specified branch's history into the current one
    • git log: show the commit history for the currently active branch
    • Github workflow:
      • git status: shows which branch you are currently on
      • git add -A: commits all files in the working branch to the staging area

JavaScript

  • JavaScript is a cross-platform, object-oriented programming language
  • Used by developers to make web pages functional and interactive
  • Crete a new folder named scripts
  • Within scripts, create a new file named main.js
  • In your index.html file, add a script element just before the closing body tag
  • script element for JavaScript is equivalent to link element for CSS
  • src attribute inside the opening script tag specifies the URL of your external .js file
  • JavaScript syntax:
    • A variable is a named container that stores a value
    • Control flow is the order in which a computer executes code in a script
    • A semicolon is required after each JavaScript statement
    • Comments are snippets of text ignored by the browser and indicated by /*, */
    • Operators are mathematical symbols that produce a result
    • Conditioinals are used to test if an expression returns true or not
    • Functions package reusable functionality and must be called to execute the code
    • Values can be passed to a function as parameters, and the function will return a value
    • A function declaration consists of:
      • function keyword
      • name of function
      • a list of parameters to the function, enclosed in parentheses and separated by commas
      • the statement that defines the function, followed by a semicolon, and enclosed in curly braces
    • The following functions are built into the browser:
      • document.querySelector(""); grabs a reference to your heading, and stores it in a variable named myHeading
      • window.alert("message"); makes a pop up box appear inside the browser window
      • console.log("message"); outputs a message to the console
    • Event handlers are code structures that listen for activity in the browser, and run code in response
    • Event handlers are attached to HTML elements for eg. a click event is triggered by the browser when the user clicks the element
